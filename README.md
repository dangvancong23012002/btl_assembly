## Đề bài

## Tệp chương trình 
- btl.cpp -> Tệp chương trình chính
- dir.asm -> Tệp tạo thư mục(tên thư mục vào từ bàn phím)
- osum.asm -> Tệp tổng lẻ dãy số nguyên
- com.asm -> Tệp cổng COM
- exit.asm -> Tệp thoát chương trình 
## Lệnh chạy
dich btl.cpp osum.asm exit.asm
